## Computer Craft

### Install PackMan

*([packman repo](https://github.com/lyqyd/cc-packman))*

`$` `pastebin run 4zyreNZy`

### Add This Repository to PackMan

`$` `edit /etc/custom-repolist`
```
/etc/custom-repolist
--------------------

robco https://raw.githubusercontent.com/the-ivo-robotnic/robco-packages/master/computercraft/packlist
```
